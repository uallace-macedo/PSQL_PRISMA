# 🐘💎 PSQL_PRISMA

Projeto para entender o uso de **ORMs `(Prisma)`** com Node e PostgreSQL.

---

## 📝 Tecnologias

### 📦 Básicas

- 🟢 `Node.JS`
- 🛣️ `Express.js`
- 🌐 `Cors`
- 🔑 `Dotenv`
- 🛠️ `Nodemon`

### 💡 Especificas

- 💎 `prisma`: interface para desenvolvimento - gerencia comandos no terminal
- 🔮 `@prisma/client`: gerencia comunicação com o banco de dados

### 🧩 Banco de Dados

- 🐘 `PostgreSQL`
